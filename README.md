# HCMC Metro Ticket Vending Machine System

#Project Overview
A comprehensive Ticket Vending Machine (TVM) system for Ho Chi Minh City Metro Line 1 (Ben Thanh - Suoi Tien). This system supports both credit card and digital wallet payments for single-trip metro tickets.

## Features
- **Destination Selection**: All 14 stations on Ben Thanh - Suoi Tien line
- **Payment Methods**: Credit Card (Visa/Mastercard) and Digital Wallets (MoMo/VNPay)
- **Ticket Generation**: QR code paper tickets with 90-minute validity
- **Multi-language Support**: Vietnamese and English interfaces
- **Error Handling**: Comprehensive error management and user notifications
- **Admin Features**: Remote monitoring and fare management
